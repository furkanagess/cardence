# Cardence – App Icon Visual Prompts (Detailed, English)

Each prompt describes a **single graphic or illustration** that combines the letter **C** and a **card (business card)** shape. The resulting image is intended to **be used as the app icon** (e.g. exported at 1024×1024, placed in the app icon asset). Prompts are written in detailed English so the visual can be generated clearly; the phrase “to be used as an app icon” defines the purpose, not a specific “app icon style” filter.

---

## Shared rule: C + card in one visual

- Every prompt describes one image where **the letter C** and **a card (business card) form** appear together in a single composition.
- The C can form the card, the card can frame the C, or both can merge into one mark.
- One clear, memorable mark: **Cardence** = **C** + **card**. The final image will be used as the app icon.

---

## 1. C formed by the card (card outline becomes C)

**Detailed prompt (EN):**

A minimal 2D graphic: the letter C is formed entirely by the outline of a single business card. The card is shown as one continuous shape — for example a curved or gently folded card — and that contour reads clearly as the letter C. Single continuous stroke or thin outline, no fill inside the C, no gradients. Colour: solid blue (#2196F3) for the shape on a pure white (#FFFFFF) background. No other shapes, no text, no shadows. The composition is centered in a square frame with enough padding so the mark stays clear when scaled down. The image is a standalone logo-style graphic intended to be used as an app icon, so it must read clearly at small sizes and work on a plain background.

---

## 2. Card as frame, C as main element inside

**Detailed prompt (EN):**

A minimal 2D graphic: a horizontal business card rectangle (with slightly rounded corners) acts as a frame or window. Inside this rectangle, the letter C is the main and only element — large, centered, and drawn in the same minimal style (e.g. outline or solid). The card outline and the C use the same line weight and the same colour so they feel like one design. Colour: solid blue (#2196F3) for both card and C on a pure white (#FFFFFF) background. No other graphics, no text, no decorative details. The whole composition is centered in a square frame with balanced padding. This graphic is intended to be used as an app icon; it should stay legible and recognizable when displayed at small sizes (e.g. on a home screen).

---

## 3. C and card merged into one mark (C wraps or supports the card)

**Detailed prompt (EN):**

A minimal 2D graphic where the letter C and a small business card shape are merged into a single logo. For example: the C wraps around one edge of the card, or the card sits inside the opening of the C, or the card leans against the C so they share a common edge — the exact arrangement is one unified symbol, not two separate objects. Single colour for both elements, consistent line weight or fill. Colour: solid blue (#2196F3) on a pure white (#FFFFFF) background. No text, no extra decoration, no shadows or gradients. The mark is centered in a square frame with even padding. The result is a clear, scalable graphic intended to be used as an app icon, readable at small sizes.

---

## 4. C with one stroke as a card (shared edge)

**Detailed prompt (EN):**

A minimal 2D graphic of the letter C in a clean sans-serif or geometric form. One of the two strokes that define the C (e.g. the outer or inner curve) is simultaneously the edge of a thin business card shape — so the card and the C share one long edge or curve, and the card appears as a flat rectangle or strip attached to the C. The result is one coherent shape: you see both the letter C and a card. Single colour, flat design. Colour: solid blue (#2196F3) for the entire shape on a pure white (#FFFFFF) background. No text, no extra elements, no shadows. Centered in a square frame with balanced padding. This graphic is intended to be used as an app icon and must remain clear and recognizable when scaled down.

---

## 5. Card rectangle with C in one corner (corner mark)

**Detailed prompt (EN):**

A minimal 2D graphic: a single horizontal business card rectangle (slightly rounded corners) is the main frame. In one corner — for example the top-right or bottom-left — the letter C sits as a clear, bold mark (like a corner stamp or watermark). The C is sized and positioned so it balances with the card and does not overwhelm it; both card and C use the same colour and style. Colour: solid blue (#2196F3) for both elements on a pure white (#FFFFFF) background. No other shapes, no text, no gradients or shadows. The composition is centered in a square frame with even padding. The image is a standalone graphic intended to be used as an app icon and must read well at small sizes.

---

## 6. Two card shapes arranged to form the letter C

**Detailed prompt (EN):**

A minimal 2D graphic made only of two thin rectangular shapes that suggest business cards. The two cards are arranged (e.g. overlapping, angled, or one curved) so that together they form the letter C — the negative space or the outlines create a clear C. Geometric and minimal; no extra curves or decoration. Single colour for both card shapes. Colour: solid blue (#2196F3) on a pure white (#FFFFFF) background. No text, no other elements. The composition is centered in a square frame with balanced padding. This graphic is intended to be used as an app icon; the C and card association should remain clear when the image is displayed at icon size.

---

## 7. C cut from card or card cut from C (negative space)

**Detailed prompt (EN):**

A minimal 2D graphic using negative space: either (a) the letter C is cut out of a solid business card shape (so you see a C-shaped hole in the card), or (b) a card shape is cut out of the letter C (so you see a card-shaped hole in the C). One dominant shape, one clear negative-space shape; the result is a single memorable mark that reads as both C and card. Flat design, no gradients. Colour: solid blue (#2196F3) for the solid part on a pure white (#FFFFFF) background. No text, no extra graphics, no shadows. Centered in a square frame with even padding. The image is intended to be used as an app icon and must stay recognizable and sharp at small sizes.

## General rules (all prompts)

- **C + card:** Every prompt describes one image where the letter C and a card form appear together in a single, unified composition.
- **Output use:** The generated visual is intended to **be used as the app icon** (e.g. square format, 1024×1024 or 512×512, clear at small sizes, no text).
- **Style:** Flat, minimal, logo-like; one dominant mark, no photos or complex illustration.
- **Colour:** Primary blue #2196F3 for the mark; background pure white #FFFFFF or very light grey.
- **Avoid:** Additional text, people, faces, photography, busy detail.
- **Brand:** Cardence = **C** + **card**; the visual should convey this in one clear symbol.
